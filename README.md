# HappyMinecraft – Server Resourcepack

Dieses Repository enthält das **Server-Resourcepack** für den HappyMinecraft RPG-Server.

## Server-Link (immer neueste Version)
https://github.com/harisjade-bambelbie/HappyMinecraft_resourcepack/releases/latest/download/HappyMinecraft_Server_resourcepack.zip

## Hinweis zu Updates
- Bei jedem Update neues Release erstellen und die ZIP **immer gleich** benennen: `HappyMinecraft_Server_resourcepack.zip`
- In der `server.properties` danach **SHA1 aktualisieren** (`resource-pack-sha1=`)

## Enthält
- RPG-Zonen-Sounds (Namespace `rpg:`)
- Vegas GUI Assets (gemerged)
